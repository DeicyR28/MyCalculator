# Python Calculator Application

This is a command-line calculator built in Python that performs basic
operations such as addition, subtraction, multiplication, and division.
The application automatically keeps a history of every operation you execute and its saved and loaded each time you start the calculator.

Features:
- Perform basic math operations by giving commads when prompted as fallow:
    Add 1 1
    Subtract 5 10
    Muliply 5 5
    Power 2 3
    Divide 6 3

- Command-line interface
- History of calculations

Technologies:
- Python
- pytest
- Git

Installation Instructions

1. Clone the repository
git clone https://github.com/username/project-name.git

2. Navigate to the project folder
cd project-name

3. Install dependencies
pip install -r requirements.txt

To Run the calculator
python main.py

To Run test 
pytest
